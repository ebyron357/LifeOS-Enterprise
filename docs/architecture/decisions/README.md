# Architecture Decision Records

This directory contains all Architecture Decision Records (ADRs) for LifeOS Enterprise.

ADRs capture significant architectural decisions, their context, options considered, and the rationale for the chosen direction.

## How to Create a New ADR

Use the [ADR template](../../../templates/adr-template.md).

ADR numbering is sequential. ADR status values: `Proposed` → `Accepted` | `Rejected` | `Superseded`.
