# Obsidian Setup

Purpose: Open this repository as the working LifeOS vault.

## Success Check

Obsidian should open directly to:

`Command Center/Daily Command Center.md`

That note should show active projects, deadlines, waiting items, reviews due, and learning due once Dataview is enabled.

## Required Community Plugins

Install these first:

1. Dataview
2. Templater
3. Calendar
4. Periodic Notes
5. Tasks
6. Obsidian Git

## Recommended Core Settings

### Files and Links

- New link format: Relative path to file
- Use Wikilinks: On
- Default location for new notes: `Inbox`
- Detect all file extensions: On

### Templates

- Template folder location: `templates`

### Daily Notes

- New file location: `Daily Notes`
- Date format: `YYYY-MM-DD`

## Opening the Vault

1. Clone or download this repository.
2. Open Obsidian.
3. Click `Open folder as vault`.
4. Select the `LifeOS-Enterprise` folder.
5. Open `Command Center/Daily Command Center.md`.
6. Enable community plugins.
7. Install Dataview.
8. Reload the dashboard note.

## What Good Looks Like

- The Daily Command Center opens without broken note paths.
- Dataview code blocks render tables instead of plain code.
- Project notes under `Projects/` appear in the Active Projects section.
- Business notes under `Businesses/` appear in review dashboards.
- New ideas go to `Inbox/` before being processed.

## Rule

Do not create random loose notes. Capture into Inbox, then process into the correct folder using a template.
