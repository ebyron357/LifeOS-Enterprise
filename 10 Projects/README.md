# Projects

Projects are outcomes with a clear finish line.

Create projects from `99 Templates/Project.md`.

Every active project must have:

- a desired outcome;
- success criteria;
- one current next action;
- an owner;
- a priority;
- an area link;
- a review date;
- a due date when appropriate.

Completed projects move to `90 Archive/Projects/` after the final review.
