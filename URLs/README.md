# URLs

Store useful links with purpose, category, tags, related tool, related project, notes, and review date.

## Extracted URL Standard

URLs extracted from videos, transcripts, articles, courses, or AI outputs must also include:

- original source record;
- evidence location or timestamp;
- exact versus inferred status;
- verification status;
- operating context;
- related project, area, goal, client, business, nonprofit, community, or research record when applicable;
- next action;
- review date.

Never construct or guess an exact URL from a tool, product, company, book, or website name. Store the named resource with `needs-verification` until the exact URL is confirmed.
