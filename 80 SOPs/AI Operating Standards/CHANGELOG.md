- Added platform-specific environment, permissions, and persistence rules.
- Added a complete document-maintenance workflow.

### Duplicate removal

Repeated language about accuracy, status, alternatives, tool selection, accessibility, and standing-rule updates was consolidated into one authoritative section per document.

### Conflict resolution

- “Continue until complete” is limited to the active authorized run or a verified persistent system.
- Scheduled tasks cannot be described as uninterrupted execution unless persistence is real.
- External writes remain approval-gated unless the explicit task authorizes them.
- GitHub remains the durable source of truth; Obsidian remains the main LifeOS interface.

### Quality control

All six documents received separate structural and semantic QC reports. Final package status: **Ready for Review** pending repository review and merge.
