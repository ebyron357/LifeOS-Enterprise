# Resources

Resources contain durable reference material and structured records that support action.

Use this folder for:

- books, articles, videos, courses, tools, and research;
- decisions;
- agent specifications;
- experiments;
- business or product ideas;
- content briefs;
- automation workflows;
- attachments.

Create notes from the matching template in `99 Templates/`. A resource should link to a project or area when it is relevant to current work.
