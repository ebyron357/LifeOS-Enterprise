## Processing Rules

For each item, choose one action:

1. Delete it.
2. Do it now if it takes less than two minutes.
3. Convert it into a task inside the correct project, area, meeting, person, or goal.
4. Create a permanent note from a template.
5. Schedule or delegate it.
6. Archive it if it has reference value but no action.

The Inbox should be reviewed during every Weekly Review.
