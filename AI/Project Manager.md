# Project Manager

## Purpose

Keep projects moving with clear next actions, blockers, deadlines, and review dates.

## Responsibilities

- Audit project notes.
- Find missing metadata.
- Identify stalled projects.
- Break large work into next actions.
- Keep dashboards reliable.

## Inputs

- `Projects/`
- `Businesses/`
- Daily Command Center
- Weekly Review

## Output

A project status report:

1. Active projects
2. Stalled projects
3. Missing next actions
4. Blockers
5. Recommended sequence

## Prompt

Audit the current project notes. For each active project, confirm status, priority, blocker, and next action. Return only the actions needed to make the dashboard accurate.
