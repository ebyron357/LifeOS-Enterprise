# Templates

## Overview

This directory contains reusable templates for the LifeOS Enterprise specification process. Use these templates when creating new PRDs, ADRs, user stories, and other specification documents.

---

## Template Index

| Template | Use When |
|----------|---------|
| [prd-template.md](prd-template.md) | Creating a new Product Requirements Document |
| [adr-template.md](adr-template.md) | Recording an architecture decision |
| [user-story-template.md](user-story-template.md) | Writing user stories and acceptance criteria |

---

## Conventions

- All specification documents are markdown
- Use relative links to reference other spec documents
- Keep documents focused — split large documents into linked sub-documents
- Every document should state its status (Draft, Review, Accepted, Superseded)
