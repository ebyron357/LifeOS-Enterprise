# Templates

This is the only canonical template folder.

## Core Templates

- Daily Note
- Weekly Review
- Monthly Review
- Project
- Area
- Goal
- Meeting
- Person
- Decision
- Resource
- SOP

## Business and AI Templates

- Agent Specification
- Experiment
- Business or Product Idea
- Content Brief
- Automation Workflow

Configure Obsidian Templates to use `99 Templates` and Daily Notes to use `99 Templates/Daily Note`.

Do not edit templates from the Properties panel when template variables are being overwritten. Use Source mode for template maintenance.
